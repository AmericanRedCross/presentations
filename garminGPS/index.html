How to take a point

Turn it on

Go to Waypoint

Take the Point

Name the Point

Go to Waypoint Manager

Rename the Point

How to take a track

Tracks = Lines

Go to Setup

Go to Tracks

Under Tracklog, Click Record, Show on Map

Return to Main Screen

Go to Track Manager

What do I do when I'm done?

Click Current Track

Click Save Track

Name the track

What if I need to create a new track?

Just make sure the tracklog is on...

... and save the track when you're done.

How do I get the data out?

Plug the GPS in to your computer

Open the Garmin Folder

Open the GPX Folder

Copy - Paste to Your Computer

Send to your Supervisor

How do I see my GPS Data?

Drag + Drop Into Google Earth

Or go to Tools>GPS>Import

Thgen click "Import From File"

And select your file
