Download OSM Data

Tips for downloading data from HOT Exports.

  1. Go to HOT Exports and login
    • If you don’t have an account already, create one
  2. Create a New Job, or use one you’ve already created
    • Name your job
    • Give it a description that makes sense to others
    • Zoom into the area of the map that you want to download
    • Click “Select Area” and draw a box around the area you want to download data from*
      Careful: If you select a giant area, you may overload the server. Do big downloads in pieces, if necessary
    • Click “Save”
  3. If you want to filter information, select the appropriate preset file from “Select Preset File"
  4. If you don’t know what this means or don’t want to filter information, click “Save”
  5. Wait for the server to process your export
  6. When the export is ready, you should see 8 different types of files you can download under “Runs”
    • Select the file type you want. Usually this will be “ESRI Shapefile”
  7. The data will download as a ZIP file
  8. Extract the data from the ZIP file. You will have a separate shapefile for the points, lines, polygons, and roads. You can now use this data in desktop GIS software like QGIS
